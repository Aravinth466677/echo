# Google Maps Integration Guide\n\n## Overview\nThe InteractiveMap component supports both Google Maps integration and a fallback coordinate-based map interface.\n\n## Google Maps Setup (Optional)\n\n### 1. Get Google Maps API Key\n1. Go to [Google Cloud Console](https://console.cloud.google.com/)\n2. Create a new project or select existing one\n3. Enable the \"Maps JavaScript API\"\n4. Create credentials (API Key)\n5. Restrict the API key to your domain for security\n\n### 2. Add Google Maps Script to HTML\nAdd this script tag to your `public/index.html` file:\n\n```html\n<script async defer\n  src=\"https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&libraries=geometry\">\n</script>\n```\n\nReplace `YOUR_API_KEY` with your actual Google Maps API key.\n\n### 3. Environment Variable (Optional)\nAdd to your `.env` file:\n```\nREACT_APP_GOOGLE_MAPS_API_KEY=your_api_key_here\n```\n\n## Fallback Map Interface\n\nIf Google Maps is not available, the component automatically falls back to a coordinate-based interface that:\n\n- Shows a visual grid for reference\n- Allows click-to-select functionality\n- Displays both reporter and issue locations\n- Provides coordinate feedback\n- Works completely offline\n\n## Features\n\n### With Google Maps:\n- Full interactive map with satellite/street view\n- Accurate location markers\n- Real-time map updates\n- Professional map interface\n\n### With Fallback Interface:\n- Simple coordinate-based selection\n- Visual grid for reference\n- Click-to-select functionality\n- No external dependencies\n- Works in all environments\n\n## Usage in Components\n\n```jsx\nimport InteractiveMap from './InteractiveMap.jsx';\n\n<InteractiveMap\n  onLocationSelect={(location) => {\n    console.log('Selected:', location);\n  }}\n  selectedLocation={issueLocation}\n  reporterLocation={reporterLocation}\n  disabled={false}\n  height={300}\n/>\n```\n\n## Customization\n\nThe map component can be customized by:\n- Adjusting the height prop\n- Modifying the default center coordinates\n- Changing the zoom level\n- Customizing marker icons\n- Styling the fallback interface\n\n## Security Notes\n\n- Always restrict your Google Maps API key to specific domains\n- Consider implementing usage quotas\n- Monitor API usage in Google Cloud Console\n- The fallback interface works without any external API calls
